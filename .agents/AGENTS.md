# Project Rules

## Visual Assets & Illustration Guidelines

Boss specifically prefers images with the following exact style. ALL AGENTS must
use this prompt/style when generating or selecting images for this project:

**Prompt Style:** "A high-end, premium minimalist architectural wireframe
sketch, monochrome, pencil on pure white background, NO vignette, NO borders, NO
drop shadow. Clean, Swiss-design inspired, lots of negative space."

**Key rules:**

- Never generate 3D renders, vector art, colorful illustrations, or generic AI
  slop.
- Images must have a pure white background so they blend seamlessly into the
  .center or hero sections.
- NEVER add drop shadows, thick borders, or vignettes to images.
